Challenge-Alura-Store
<h1 align="center"> Challenge-Alura-Store </h1>  

## Descripción:
Durante este desafío se analizaran datos de ventas, rendimiento y reseñas de las 4 tiendas de Alura Store. El objetivo es ayudar al Sr. Juan a determinar cual es la tienda menos eficiente basandose en el analisis previo.

## Estado del proyecto:
:checkered_flag: Proyecto finalizado :checkered_flag:

## :gear:Descripción del proyecto:
1. Recolección y preparación de datos:
   Se importaron los archivos de ventas de las cuatro tiendas y las librerias necesarias.
2. Análisis exploratorio:
   Se examinaron los DataFrame de cada tienda para conocer el contenido de cada columna y la cantidad de registros.
3. Análisis de facturación:
   Mediante una función se calculó el ingreso total de cada tienda y se utilizó un grafico de barras para visualizar el resultado.
4. Análisis de ventas por categoría de productos:
   Por medio de una función se ordenaron las categorías de productos de mayor a menor en base a la cantidad vendida por tienda. Luego se empleo un grafico de torta para observar la distribución.
5. Análisis de la calificación del cliente:
   Se creó una función para determinar la evaluación promedio de los clientes por tienda y se trazó un grafico de dispersión para evaluar los resultados.
6. Análisis de los productos más y menos vendidos:
   A través de 2 funciones se estableció cual era el TOP 3 de los productos más y menos demandados por tienda. Se dispuso un grafico de barras horizontales para apreciar los resultados. Además se emplearon estos     mismos datos para evaluar la diversificación de las ventas por sucursal.
7. Análisis del costo de envío:
   Se calculó el costo de envío promedio por tienda y luego se grafico un histograma para evaluar los resultados.
8. Análisis del desempeño geografico:
   Tomando como parametros principales los datos de longitud y latitud presentes en los DataFrames, se creó una función que retorna un mapa de calor que nos permite visualizar las calificaciones de los clientes     y los ingresos por zonas geograficas.
9. Normalización, ponderaciones y puntuaciones finales:
   Con el fin de comparar adecuadamente los resultados obtenidos en los pasos anteriores, se prodeció a realizar una normalización. Luego, se definieron las ponderaciones para cada criterio y por último se ejecutaron los calculos para determinar la puntuación total de cada tienda.
10. Conclusiones finales:
    Aquí se exponen las conclusiones finales y se fundamenta la recomendación de cual es la tienda que debe vender el Sr. Juan para iniciar su nuevo emprendimiento.
